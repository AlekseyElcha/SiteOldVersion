• Как можно отправить нам сообщение онлайн?
∗ Telegram-бот: Перейдите по ссылке и следуйте инструкциям бота.
Нажмите, чтобы показать/скрыть подробную инструкцию по использованию Telegram-бота

IOS

Инструкция для пользователей Android:
1) Перейдите по ссылке.
2) Если браузер запросит переход в Telegram - разрешите ему это сделать.


3) После того, как откроется чат с ботом, нажмите на кнопку START или наберите и отправьте /start.
4) Следуйте указаниям бота в чате.

Android

Инструкция для пользователей Android:
1) Перейдите по ссылке.
2) Если браузер запросит переход в Telegram - разрешите ему это сделать.

4) Следуйте указаниям бота.

ПК

Инструкция для Telegram Desktop и Telegram Web на компьютере:
1) Перейдите по ссылке.
2) Откройте Telegram - в браузере (кнопка «Open in Web») или в приложении (открыть Telegram Desktop).


3) Чтобы запустить бота, нажамите на кнопку «Меню» или «Menu», затем нажмите на «Запустить бота» или отправьте сообщение /start.

4) Следуйте указаниям бота в чате. Важное примечание! Кнопки взаимодействия открываются по доп.кнопке справа!
